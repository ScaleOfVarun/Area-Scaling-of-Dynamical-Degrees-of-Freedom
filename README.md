# Area-Scaling-of-Dynamical-Degrees-of-Freedom

Code to reproduce the figures in the paper **“Area-Scaling of Dynamical Degrees of Freedom”**.

This repository contains a Jupyter notebook that runs the numerical experiments and generates the plots used in the manuscript, along with supporting data and figure output folders.  
(Please add the arXiv/DOI link once available.)

---

## Repository contents

- `area_scaling_num_expt.ipynb` — main notebook to run experiments and generate figures  
- `data/` — input data and/or cached outputs used by the notebook  
- `figures/` — generated figures (PDF)
